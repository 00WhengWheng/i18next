i18next - translation made easy

i18next is a full-featured javascript library for translating your webapplication.

Production (min) Development (full)

node.js:

npm install i18next
more information (npm module)

Benefits

  1. support for variables
  2. support for nesting
  3. support for context
  4. support for multiple plural forms
  5. detect language
  6. graceful translation lookup
  7. jquery function
  8. get string or object tree
  9. get resourcefiles from server
  10. post missing resources to server
  11. highly configurable

  12. something missing?

Usage Sample

page source:

<!DOCTYPE html>
<html>
  <head>
    <script type="text/javascript" src="[PATH]/jquery.js" />
    <script type="text/javascript" src="[PATH]/i18next.js" />
  </head>
  <body>
    <ul class="nav">
      <li><a href="#" data-i18n="nav.home"></a></li>
      <li><a href="#" data-i18n="nav.page1"></a></li>
      <li><a href="#" data-i18n="nav.page2"></a></li>
    </ul>
  </body>
</html>

loaded resource file (JSON):

{
  "app": {
    "name": "i18next"
  },
  "nav": {
    "home": "Home",
    "page1": "Page One",
    "page2": "Page Two"
  }
}

javascript code:

i18n.init(function(t) {
  // translate nav
  $(".nav").i18n();

  // programatical access
  var appName = t("app.name");
});
